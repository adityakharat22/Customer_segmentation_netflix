# Customer Segmentation on Netflix Dataset

## Project Overview

This project focuses on **Customer Segmentation** using clustering techniques on a Netflix dataset(this dataset is just for learning purpose).
The goal is to group users based on their behavior, engagement, and content preferences to help businesses make better decisions.

---

##  Objectives

* Understand customer behavior
* Segment users into different groups
* Improve targeted marketing strategies
* Analyze engagement and churn patterns

---

## 📂 Dataset

* Dataset contains information about Netflix users
* Includes features like:

  * User activity
  * Engagement level
  * Content preferences

---

##  Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

---

## Machine Learning Techniques

* Data Preprocessing
* Feature Scaling
* K-Means Clustering
* Elbow Method (for optimal K selection)

---

## Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Selection
4. Scaling the data
5. Applying K-Means Clustering
6. Finding optimal clusters using Elbow Method
7. Visualization of clusters
8. Exporting segmented data

---

##  Results

* Successfully segmented users into:

  * Engagement clusters
  * Churn clusters
  * Content preference clusters
* Generated meaningful insights from user behavior

---


---

##  Key Insights

* Users can be grouped based on activity and preferences
* High engagement users behave differently from low engagement users
* Clustering helps in identifying potential churn users
